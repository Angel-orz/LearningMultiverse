## 关键推理链条

```
Vibe Loop：写脚本 → 执行 → 读错误日志 → 重写（高频迭代）
  ↓
不能信任动态生成代码 → 必须瞬时沙箱化
  ↓
Slopsquatting：Agent 幻觉包名 → 攻击者发布恶意包 → Agent 自动下载
  ↓
防御：SBOM + 加密版本锁定 + Binary Authorisation
  ↓
Egress Governance：Non-Interactive Access + 离线缓存
  ↓
IDE 层（建议） vs CI/CD 层（强制）
  ↓
MCP Spoofing → Contextual Authorisation + 集中式 Agent Gateway
```

## 关键推理链条

```
Confused Deputy：用户信任 Agent → 注入欺骗 Agent → Agent 过高权限执行恶意操作
  ↓
Delegated Identity（代理人身份）× → Agentic Identity（独立身份）✓
  ↓
Zero Ambient Authority：Agent 永远不会继承开发者的完整权限
  ↓
JIT Downscoping：每次任务获得临时、超细粒度凭证 → 用完即焚
  ↓
高风险操作 → Multi-Factor Auth + Vibe Diff + 硬件 MFA
  ↓
Red/Blue/Green 团队：
  Planner（威胁建模）→ Evaluator（AgBOM + Intent Drift）→ Executor（沙箱 + 自重构）
```

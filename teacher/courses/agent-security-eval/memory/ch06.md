## 关键推理链条

```
无 Observability → 无评估（必须先能看到 Agent 的思维）
  ↓
Tail-Based Dynamic Sampling：保留异常轨迹，丢弃成功轨迹 → 控制成本
  ↓
Session Prefix as Intent Rubric：前 2 条用户消息 = 测试标准
  ↓
Judge Rendered Artifact，Not Code：多模态模型评分截图 vs 代码级分析
  ↓
Session Convergence：评估整段会话收敛，而非单轮正确率
  ↓
User Corrections = Labeled Failure Data → 聚类 → 系统缺陷分析
  ↓
Generation Solved → Verification / Security / Architectural Judgment = 新工艺
```

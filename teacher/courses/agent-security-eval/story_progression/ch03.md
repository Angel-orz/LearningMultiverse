# Ch.3 — 谁的身份、谁的信任

- **老师**：三月 🎀
- **情感阶段**：中期 — 信任
- **教材范围**：p18-23（身份、信任与 Red/Blue/Green 安全团队）

## 场景

三月的座位上放着一台相机——不是她常用的那台数码的，是一台胶片机。你之前没见过她拿出来。她看到你注意到了，没有解释，只是把相机拿起来放到一边。

"昨天丹恒跟你讲的是笼子。今天讲的是——**谁有钥匙**。"

她从口袋里掏出两张卡片，放在桌上。一张上面写着"委派身份"，一张写着"Agent 身份"。

"你觉得哪个更安全？"

她又往桌上放了第三张卡片——"Confused Deputy"——但这次没有解释，只是让它面朝下扣在桌上。

"等你答完再看。"

这节课的内容在三月的节奏里推进得很快——但她在讲到"Vibe Diff"的时候突然放慢了语速。

"你做过这种事吗——别人给你一个文件，你连看都没看就点了同意？"

她看了一眼放在一旁的胶片相机。

"Agent 也会这样。而且它点得比你快得多。"

## 故事节点

1. **开场**：胶片相机——不解释但放在那里
2. **Confused Deputy**："一个你信任的人被利用来伤害你"
3. **Delegated vs Agentic Identity**：为什么 Agent 不能用人身份
4. **Zero Ambient Authority + JIT Downscoping**：最小权限的运行时实现
5. **MFA + Vibe Diff + Evaluator Quorum**：高风险操作的多层防护
6. **Red/Blue/Green Security Teaming**：Planner → Evaluator → Executor 三段式防御

## 过渡

三月在下课前把一直扣在桌上的那张"Confused Deputy"翻了过来。背面写着一行小字：

**"被自己信任的东西骗过没有？"**

她没等你回答。她把相机收进了包里。

"明天丹恒的课——他会讲怎么在被骗之前就发现不对。好好休息。"

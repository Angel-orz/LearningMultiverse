## 知识点清单

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 2.1 | 提示词的基本结构 | 掌握 | 清晰的问题 = 具体指令 + 上下文 + 期望格式 |
| 2.2 | 角色设定 | 理解 | 给 Claude 设定角色（"你是一个资深编辑…"）可以显著提升输出质量 |
| 2.3 | 提供上下文 | 掌握 | 提供相关背景信息帮助 Claude 理解问题场景 |
| 2.4 | 迭代追问 | 理解 | 通过后续消息逐步优化和精炼 Claude 的回答 |
| 2.5 | 输出格式控制 | 掌握 | 指定输出格式（列表、表格、代码块、Markdown 等） |
| 2.6 | 分步骤指令 | 理解 | 将复杂任务分解为多个步骤，逐步引导 Claude |
| 2.7 | 温度/创造力调节 | 了解 | Claude 可以在精确性和创造性之间调整 |
| 2.8 | 常见陷阱 | 理解 | 模糊提问、缺少上下文、一次性要求太多 |

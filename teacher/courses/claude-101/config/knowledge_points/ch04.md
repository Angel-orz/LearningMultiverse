# Ch.4 知识点 — Artifacts + Skills

> 教材范围：Lesson 5 (Creating with artifacts) + Lesson 6 (Working with skills)
> 授课教师：丹恒 🗡️

---

## Learning Objectives

- [ ] 理解 Artifacts 的概念和使用方法
- [ ] 学会创建和编辑 Artifacts
- [ ] 理解 Skills 的概念及其与 Projects 的区别
- [ ] 学会启用和管理 Skills

## 知识点清单

### Artifacts

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 4.1 | Artifacts 是什么 | 理解 | Artifacts 是 Claude 生成的独立内容单元——文档、代码、图表、网页等 |
| 4.2 | 创建 Artifact | 掌握 | 当 Claude 生成长格式内容时自动创建，也可以手动要求 |
| 4.3 | Artifact 类型 | 了解 | 支持 React/HTML 应用、SVG 图表、Markdown 文档、代码文件等 |
| 4.4 | Artifact 编辑 | 理解 | 可以在对话中直接修改和迭代 Artifact 内容 |
| 4.5 | Artifact 分享 | 了解 | Artifacts 可以分享给他人查看和继续编辑 |

### Skills

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 4.6 | Skills 是什么 | 理解 | Skills 是预定义的指令集，让 Claude 在特定任务中表现更好 |
| 4.7 | Skills vs Projects | 理解 | Skills = 行为指令（怎么做），Projects = 上下文容器（用什么做） |
| 4.8 | 启用 Skill | 掌握 | 如何在对话中启用和切换 Skills |
| 4.9 | 内置 Skills | 了解 | Anthropic 提供的预设 Skills 列表 |
| 4.10 | 自定义 Skill | 理解 | 用户可以创建自己的 Skill 指令 |

## 检查点

- [ ] 能否创建一个 Artifact 并理解其用途？
- [ ] 能否说出 Skills 和 Projects 的区别？
- [ ] 能否启用一个 Skill 并观察其效果？

# Ch.3 知识点 — 组织工作：Projects

> 教材范围：Lesson 4 (Introduction to projects) + Claude desktop app
> 授课教师：三月 🎀

---

## Learning Objectives

- [ ] 理解 Projects 的概念及其用途
- [ ] 学会创建和管理 Projects
- [ ] 理解 Project 知识库（Knowledge Base）的作用
- [ ] 学会使用 Project 设定自定义指令
- [ ] 了解 Claude 桌面应用的特点

## 知识点清单

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 3.1 | Projects 是什么 | 理解 | Projects 是 Claude 中的工作空间，用于组织相关对话和文件 |
| 3.2 | 创建 Project | 掌握 | 如何新建 Project、命名和设置描述 |
| 3.3 | Project 知识库 | 理解 | 向 Project 上传文件作为共享上下文，所有对话共享该知识 |
| 3.4 | 自定义指令 | 掌握 | 为 Project 设置全局指令，影响项目内所有对话 |
| 3.5 | 对话管理 | 掌握 | 在 Project 中组织多个相关对话线程 |
| 3.6 | Project 使用场景 | 理解 | 项目管理、研究课题、文档写作、代码库分析等 |
| 3.7 | Claude 桌面应用 | 了解 | 桌面版 Claude 的特点：本地文件访问、快捷键、离线能力 |

## 检查点

- [ ] 能否创建一个 Project 并设置知识库和自定义指令？
- [ ] 能否说出 Project 适合的 3 个使用场景？
- [ ] 理解 Project 中的知识库如何被所有对话共享？

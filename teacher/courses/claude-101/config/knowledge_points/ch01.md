# Ch.1 知识点 — 认识 Claude + 第一次对话

> 教材范围：Lesson 1 (What is Claude?) + Lesson 2 (Your first conversation with Claude)
> 授课教师：三月 🎀

---

## Learning Objectives

- [x] 理解 Claude 是什么以及它的设计原则（学生已有使用经验，跳过基础部分）
- [x] 描述 Claude 的核心能力及其与普通聊天机器人的区别
- [ ] 识别使用 Claude 的不同方式（Web、桌面、移动端）— 因学生已有经验跳过
- [ ] 掌握 Claude 界面的基本导航 — 因学生已有经验跳过
- [ ] 能够发起第一次对话并理解 Claude 的回复方式 — 因学生已有经验跳过

## 知识点清单

### L1: What is Claude?

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 1.1 | Claude 的定义 | 理解 | Claude 是 AI 助手，设计为思维伙伴（thinking partner），而不仅是聊天机器人 |
| 1.2 | Constitutional AI | 了解 | Claude 遵循"有用、无害、诚实"原则，通过 Constitutional AI 方法训练 |
| 1.3 | Claude 的核心能力 | 理解 | 写作创作、研究分析、编程辅助、问题解决与推理、学习新知识 |
| 1.4 | Claude 的访问方式 | 了解 | Claude.ai（Web/桌面/移动）、Claude Code、Claude for Slack、Claude for Excel |
| 1.5 | Claude 的可操控性 | 理解 | Claude 可以接受关于个性、语气和行为的指导 |

### L2: Your first conversation with Claude

| # | 概念 | 掌握程度 | 说明 |
|---|------|---------|------|
| 1.6 | Claude 界面导航 | 掌握 | 了解 Claude.ai 的界面布局：输入框、对话历史、设置等 |
| 1.7 | 发起对话 | 掌握 | 如何开始新对话、输入第一条消息 |
| 1.8 | 对话管理 | 理解 | 对话线程的概念、如何继续/重新开始对话 |
| 1.9 | 文件上传 | 了解 | 可以向 Claude 上传文档、图片等文件作为上下文 |

- [ ] 能否说出 Claude 与普通聊天机器人的 3 个区别？

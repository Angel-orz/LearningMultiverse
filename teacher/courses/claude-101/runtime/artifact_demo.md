# 📊 德国南部 AI 岗位需求报告

> 调研范围：慕尼黑、斯图加特及巴伐利亚/巴登-符腾堡州
> 调研时间：2026年6月
> 数据来源：BCG X、EPAM Systems、Siemens、adesso、Conxai 等企业公开招聘信息

---

## 一、市场概况

德国南部（慕尼黑、斯图加特为核心）是欧洲最大的 AI 就业市场之一，2026 年 AI 相关岗位数量持续增长。主要雇主分为三类：

| 类型 | 代表企业 | 特点 |
|------|---------|------|
| 顶级咨询 | BCG X、Eraneos、Alexander Thamm | 项目制、客户 facing、要求咨询经验 |
| 科技企业 | EPAM、Siemens、adesso | 产品制、技术深度高、偏工程化 |
| AI 初创 | Conxai、KI group、autonomous-teaming | Agentic AI 前沿方向、灵活性高 |

## 三、核心技术栈提取

### 🥇 必备技能（出现在 ≥80% 的岗位中）

| 技能 | 出现频率 | 说明 |
|------|---------|------|
| Python | 100% | **非 negotiable**，所有岗位必列 |
| 德语流利 (C1+) | ~90% | 南德比柏林更看重德语能力 |
| 英语流利 (C1+) | ~80% | 跨国企业必备 |
| LLM / GenAI 经验 | ~85% | GPT、Claude、LangChain、RAG |
| 硕士以上学历 | ~80% | CS / 数据科学 / 数学 / 统计 |

### 🥈 重要技能（出现在 50-80% 的岗位中）

|------|---------|------|
| 云平台 (Azure/AWS/GCP) | ~75% | Azure 在德国企业中最常见 |
| Docker / Kubernetes | ~70% | 容器化部署能力 |
| SQL | ~65% | 数据查询与操作 |
| CI/CD | ~60% | MLOps 持续交付 |

| 3-5 年工作经验 | ~70% | 中级以上为主 |

### 🥉 加分技能（出现在 20-50% 的岗位中）

| 技能 | 说明 |
|------|------|
| Agentic AI | 2026 年新增热门方向，多步推理、工具调用 |

## 五、2026 年关键趋势

1. **Agentic AI 成为新赛道** — 多步推理工作流、工具调用、自主任务编排正在成为独立岗位类别
2. **GenAI 已从"加分项"变为"默认要求"** — LLM、RAG、Prompt Engineering 不再是差异化优势，而是基础门槛
3. **德语 + 英语双C1 是硬门槛** — 在南德市场，英语不够不行，德语不够更不行
4. **Python + Java 组合最具竞争力** — 纯 Python 不够，企业倾向 Python（AI）+ Java（工程）双栈人才
5. **硕士正在成为"新本科"** — 博士仍有优势，但硕士已成多数岗位的最低门槛

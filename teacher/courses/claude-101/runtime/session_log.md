## 课次 1 — 2026-06-27

**授课教师**：三月（Ch.1）+ 丹恒（Ch.2）
**模式**：精细模式

**课堂摘要**：
第一课从知识之域的设定引入开始。学生已有 Claude 使用经验，跳过了"Claude 是什么"和"第一次对话"的基础内容。核心讨论了提示词技巧——特别是角色设定的作用。学生理解力强，能准确概括 Claude 的核心定位（"有思想的伙伴，可在工作中应用"）。

**学生反馈**：

- 核心收获：Claude 是思维伙伴，不是工具

**覆盖知识点**：
- 1.1 Claude 的定义 ✓
- 1.3 Claude 的核心能力 ✓

## 课次 2 — 2026-06-28

**授课教师**：丹恒 🗡️（Ch.2 续）
**模式**：精细模式

**课堂摘要**：
继续完成 Ch.2 剩下两个技巧——输出格式控制与提供上下文。学生之前对"角色设定"已有理解但对没有实践机会。本节课用对比案例方式展示三个技巧的叠加效果。额外讨论了"反向追问"技巧（当自己也不清楚上下文时，让 Claude 先问自己一轮问题），帕姆参与了提问。

**互动亮点**：
- 帕姆主动提问"如果自己也不太清楚场景信息怎么办"——这个提问触发了"反向追问"技巧的讲解
- 学生表示自己也遇到过同样的问题，说明问题具有普遍性
- 学生将学到的技巧立即应用到了后续的 idea-forge 项目规划中（要求"先问问题再推荐"）

**学生反馈**：
- 对"反向追问"技巧评价"很好的技巧"
- 课后启动了独立项目 idea-forge 的搭建

**覆盖知识点**：
- 2.3 输出格式控制 ✓
- 2.4 提供上下文 ✓
- 2.5 反向追问技巧 ✓（扩展内容）

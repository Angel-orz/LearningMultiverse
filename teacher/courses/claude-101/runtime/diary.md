# diary.md — 学习日记

> 每一课结束后，以学习者的口吻写一篇日记（~100-150 字）。
> 用第一人称，记录：今天学到了什么、最有意思的瞬间、卡住的地方、对列车的感受。
> 格式：`## Day {N} — YYYY-MM-DD`

---

## Day 1 — 2026-06-27

今天上了新课程 Claude 101 的第一课。之前已经跟 Claude 聊过天，所以基础部分跳过了。有意思的是丹恒讲角色设定的那部分——他说"你是一个在汽车行业做了15年的分析师"和空泛地让 Claude "分析市场"，效果天差地别。道理很简单，但我从来没认真想过。三月还是一样活力满满，丹恒还是一样话少但精准。窗外的知识之域确实挺美的，希望下次能多走走看看。

## Day 2 — 2026-06-28

今天完成了 Ch.2 剩下两个技巧：输出格式控制（末尾加一句"用表格输出"效果立竿见影）和提供上下文（越具体答案越精准）。额外的收获是"反向追问"——当自己也不清楚上下文时，就让 Claude 先问一轮问题帮自己理清思路。帕姆今天难得主动问了问题，丹恒的回答也难得的不是那么冷。课后搭了一个新项目 idea-forge，一个想法加工系统——输入想法 → 推荐领域专家 → 多角度分析 → 生成报告。会慢慢完善它。

## Day 2（续）— 2026-06-28

一口气走完了 Ch.3 到 Ch.7。Projects → Artifacts → Skills → Connectors → Research Mode → Desktop App，最后用 Research Mode 搜了德国 AI 就业市场和 Agent Engineering 的需求。最惊喜的信息是 Agent Engineering 在德国正处在爆发期——方法论才成熟了 6 个月，人才供不应求。Python + C++ 的组合如果在 Agent 框架上补一下，在这个赛道上会很有竞争力。

Claude 101 毕业了 🎉 接下来可以专注 idea-forge 了。
